# composer-faye-app-integration-plugin

## Usage

Add this to your composer.json:
```
{
    "name": ".../...-integration-plugin",
    "type": "faye-app-integration-plugin",
    "require": {
        "cravler/faye-app-integration": "dev-master"
    }
}
```
